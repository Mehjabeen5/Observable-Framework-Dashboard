# Observable-Framework-Dashboard
This repository documents an interactive exploratory data analysis dashboard
developed using Observable.

## Live Dashboard
https://observablehq.com/d/24278b547f0126cd

## Overview
The dashboard explores cardiovascular disease risk factors using interactive
visualizations and filters to support exploratory analysis.

## Notes
The full implementation is hosted on Observable. This repository serves as a
portfolio reference and entry point to the live dashboard.
